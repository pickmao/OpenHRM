# OpenHRM

干部动态调整与智能预警系统

## 项目结构

- `干部动态调整系统前端/` - Vue.js 前端项目
- `干部动态调整系统后端/` - Django 后端项目
- `干部动态调整与智能预警模块需求说明.docx` - 需求文档

## 技术栈

### 前端
- Vue 3
- Vite

### 后端
- Django
- Python

## 安装与运行

### 前端
```bash
cd 干部动态调整系统前端/vue-project
npm install
npm run dev
```

### 后端
```bash
cd 干部动态调整系统后端
pip install -r requirements.txt
python manage.py runserver
```
